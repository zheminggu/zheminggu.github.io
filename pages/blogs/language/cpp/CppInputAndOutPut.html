Open File

Simple way:

						
    std::ifstream inputFile("input file path");
    std::ofstream outputFile("output file path");

Complex way:

						
    // use function   .open(file path , mode)
    std::ifstream inputFile;
    std::ofstream outputFile;
    std::inputFile.open("File path", ios::in | ios::binary)
    std::outputFile.open("File path", ios::out | ios::binary | ios::app)

how to write mode?

' | ' combines mode together

function description
ios:in open for input
ios::out open for output
ios::binary open in binary mode
ios::ate set initial position at the end of file
ios::app all output are performed at the end of file
ios::trunc delete previous content and add new one
