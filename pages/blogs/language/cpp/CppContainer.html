Cpp containers

Common functions

this section describes the meaning of each common functions to help understand the name of each function, so that easier to memorize functions in different container.

c means container, i means index, e means element

get elements

useful

Function Description
container[i] get elements at index i
.front() get first element
.back() get last element
.find(e) find element, only for maps and sets
.count(e) count the number of element (0 or 1), only for maps and sets

might use

Function Description
.at(i) equals to c[i]
.data() a pointer to first element equals to *c or c[0]

add elements

useful

Function Description
.push_back(e) push an element at the back of container
.insert(i, e) insert an element at index I

might use

Function Description
.emplace_back(args) faster way to push at back
.emplace(i, args) faster way to insert at index I
.push_front(e) push an element at the front of container

insert vs emplace 

						
class MyClass{
    int a;
    float b;
}
vector<MyClass> c;
int i= 3;
// with insert
MyClass e(1, 1.5);
c.insert(i, e);
// with emplace
c.emplace(i, 1, 1,5)  // 1, 1.5 is arguments to construct MyClass

remove elements

useful

Function Description
.erase(i) pop elements at I
.pop_back() pop elementes at the end
clear() pop all the elements

might use

Function Description
.erase_if(I, function) pop elements at I if function return true

iterate

useful

						
//for each loop 
for(auto& e: c){  }

//iterate through
for(iterator it = c.begin(); it!= c.end(); it++){  }

//without iterator
for(int i =0; i< c.size(); i++){  }
Function Description
.begin() iterator point to first element
.end() iterator point to last element

might use

Function Description
.rbegin() reversed begin = .end()
.rend() reversed end = .begin()
.cbegin() const iterator point to first element, use it when you don't want to modify element in loop
.cend() const iterator point to last element, use it when you don't want to modify element in loop
.crbegin() const reversed begin
.crend() const reversed end

check size

useful

Function Description
.size() return the number of elements in the container
.empty() check if container is empty

might use

Function Description
.capacity() return the size of the storage space currently allocated for the vector
.max_size() max number of elements vector can allocate

modify size

might use

Function Description
.shrink_to_fit() shrink capacity to fit size
.reserve(mySize) request to shrink capacity to fit mySize
.resize(mySize) shrink size to mySize

Array

c++ kind array, add some functions than normal array

feature

• cannot modify size and capacity

• .fill(): fill array with value

• access element is quick

include

						
#include <array>

init

						
//std::array<T, size> 
std::array<int, 2> a = {1,2} ;
>>a = (1,2)
a.fill(1);
>> a = (1,1)

Vector

array which can change size

feature

• very common

• insert is slow

• access element is quick

include

init

std::vector<int> a(2, 8);
>> a = (8, 8)

std::vector<int> a(2);

feature

• access element slower than unordered_set

• elements are unique

• can iterate through elements by order

Unordered_set

store unique elements in no particular order

feature

• access element quicker

• elements are unique

• iterate is difficult
