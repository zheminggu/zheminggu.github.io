Technique Details

AI Architecture

The central purpose of my AI is to mimic the human. So it has been split into Determine, Conscious, Behavior and Event.

  • Determine: Like the human, determine is our brain to handle decisions.
  • Conscious: Conscious can divide into two parts, the inside conscious (to generate instinct behaviors) and the surface conscious (to produce learned behaviors).
  • Behaviors: Instinct behavior is the behavior we born with such as moving. Learned behavior is the behavior we need to learn to combine different basic moves such as patrolling.
  • Event: Events happen in the environment

The Enemy folder is the models and prefabs of NPC, Scene is created to test the AI. AIFolderCreate is a simple folder-create application, it can create folders automately. As for the GameSet, it's some specific codes for the game.

AICharacterBrain

AICharacterBrain is one of the code files in the AI plugin. I take it just for an example of my code. It's the core of the AI system folded in the Determine to make decisions.

Region Public is variables which are public

Region Private is variables which are private

Region Initialize is to initialize different data to the brain

The Update method is to make decisions when time's up NPC will detect the environment and judge if it needs to make decisions. If it needs to make a decision, then call change decision
