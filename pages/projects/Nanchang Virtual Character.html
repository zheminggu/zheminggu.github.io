Nanchang Virtual Character

This project is to build a real-time reacting character base on the quintessence of Chinese culture -- Sichuan Opera. It was built with Unreal Engine and motion capture, coding with the blueprint.

Technique Details

Connect Unreal with the mothion capture

The first step of this project is to connect Unreal with the motion capture. It seems simple at first glance, however, in practice, various problems continuously shows up, troubles me a lot. Here is a picture I was wearing the motion capture clothes and debugging the connection

Project Architecture

I undertook the architecture design and mainly blueprint coding, to create real-time interaction. The coding part of this project consists of three parts: lighting effects, camera switching, character behaviors.

  • Lighting effects: With the experience of SCO light simulation, I built the lighting system for this project. Detail information of this system is in SCO light simulation page.
  • Camera switching: With the power of events in Unreal, using the Levelsequence can easily control the switch time of camera.
  • Character behaviors: After connecting with the motion capture, the charactere need to interact with the stage. With the knowledge of opera, I designed face changing and fire-spitting for the virtual character.
