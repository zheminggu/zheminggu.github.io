Ray Tracing

Background

This is a python project for my computer graphics final project. In this project, I use jupyter notebook to edit my program. You can download it in my github link. To better understand project you can read comments in the jupyter notebook. This program only achieved sphere interaction, for further step, you can add triangle intersection, then it would able to render arbitrary models. To speed up the program, you run it in CUDA or other GPU library, instead of render it in one core of CPU.

User guide

Install jupyter notebook

To run this python project, make sure python is installed, the go to terminal which contains python environment.

Input:

						
    pip install numpy
    pip install PIL
    pip install notebook

Run program

run jupyter notebook by input

						
    jupyter notebook

it should open a jupyter notebook website in your web browser, then, open my jupyter notebook in your web browser.

After open my jupyter notebook, click Run button, and then click Restart and Run All Cells in pop out window.

Wait until progress bar at the bottom of program move to 100%, then it would show a ray traced picture in your picture browser.

Result

Project structure

Define

Load texture: At the very beginning load model textures for each model

Define models: Then define sphere models with various parameters. These models would behave differently in final picture

Define lights: Light is also important in a scene, so after defined model define lights with different parameters.

Define parameters: define parameters such as image buffer, viewport width, viewport height, distance from camera to viewport, and camera position

Tracing ray

The main entrance of this program is actually at the vary bottom, it's in section ray tracing all the screen position.

Ray tracing all the screen position: For each pixel, from camera to viewport shoot a ray, after finish tracing show image.

Shoot a ray to get color: for each shoot ray, detect if intersect with any sphere. if intersect, get a color, and reflect ray from intersect position. Otherwise, return black.

Detect intersect (if intersect): calculate if the ray is intersect with any sphere

Calculate color (get a color): calculate color according to light parameter and intersect position and model parameter, then return color.

Show Images (show image): show image according to image buffer.
